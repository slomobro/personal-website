# Images

Put the hero photo here as **`hero.jpg`**.

The hero (`.hero__media` in `assets/css/main.css`) loads `../img/hero.jpg`.
Until that file exists, the hero shows a moody gradient fallback so nothing
looks broken.

Tips:
- A landscape/tall photo both work — the image is set to `cover` and centered.
- Keep it reasonably sized for the web (roughly 2000px on the long edge,
  saved around 80% quality) so the page stays fast.
- `.jpg` is assumed. If you use a different name or format, update the URL in
  `.hero__media`.
